# TP - Travail collaboratif sur la protection des données
Document reponse: 


* [framapad du groupe B](https://mypads2.framapad.org/mypads/?/mypads/group/pad-pppe-x81hev98e/pad/view/rgpd-groupe-2-w51hgv95t)

Le *framapad* est un document avec editeur en ligne de la societé de logiciels libre *Framasoft*.

* Commencer par *entrer votre prénom* pour vous identifier lors de votre travail d'edition.
* Rédiger votre réponse à l'une des 4 questions ci-dessous: Préciser le numero de la question.
* Une fois tous les changements terminés, marquez votre version avec une étoile pour la sauvegarder en ligne et la retrouver dans l'historique.

{{< img src="../images/frama2.png" >}}

* Enregistrer le document au format *.text* dans vos documents. Vous pourrez l'éditer de façon plus complète dans LibreOffice Writer dans une prochaine séance.

## Recherche Documentaire: Questions
Vous trouverez les reponses aux questions en parcourant les videos de la page [lespritsorcier.org](https://lespritsorcier.org/dossier-semaine/objets-connectes/):

{{< img src="../images/lespritsorcier.png" link="https://lespritsorcier.org/dossier-semaine/objets-connectes/" caption="Données personnelles, tous connectés" >}}

1. Les objets connectés me suivent à la trace. Qu'est-ce que cela signifie? Existe t-il des outils ou des logiciels pour le prouver? Peut-on les utiliser sans être espionné?
2. Mon smartphone m'espionne... Quelles sont les données qui sont capturées, et par qui? Comment faire pour limiter le problème?
3. Réseaux sociaux: Quelles informations peuvent être déduites de mon activité sur les réseaux? Peut-on conserver sa vie privée en agissant avec modération sur les réseaux?
4. Les gestes qui sauvent. Comment diffuser le moins d'informations possibles, surtout si celles-ci sont *sensibles*?
