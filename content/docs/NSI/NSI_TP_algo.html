Conseils sur le démarche de projet: Lien vers la page labri.fr

Selectionner :

Le projet choisi doit:

  • proposer un programme en plusieurs fichiers
  • au moins un de ces fichiers doit être construit comme un module
  • au moins un de ces fichiers doit présenter des tests fonctionnels
  • des tests structurels doivent être placés à différents endroits du programme
  • Le programme doit utiliser au moins UNE structure de données complexe (pile, file, tableau, autre...), et/ou un algorithme recursif
