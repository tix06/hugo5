Flash card 4

On peut vouloir convertir les données avec des valeurs exactes et non approchées. Pour eviter les confusions avec le kilobit, on utilise un autre nom pour l’unité : le Kibibit (Kibit), qui vaut 1024 bits (soit 210).

Les multiples de l’octet deviennent: Kibioctet (Kio), Mébioctet (Mio), Gibioctet (Gio), Tébioctet (Tio), Pébioctet (Pio).

Le tableau de convertion est alors le suivant :

unité valeur exacte, en octets
Kibioctets (Kio) 210
Mébioctet (Mio) 220
Gibioctet (Gio) 230
Tébioctet (Tio) 240
Pébioctet (Pio) 250 
  </ol>
</div>
