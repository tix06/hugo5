Modéliser une transformation chimique

Comment utiliser l'animation

Vous pouvez modifier, directement dans la 1ere ligne du tableau d'avancement, l'équation de réaction, avec :
  • le nom des espèces chimiques
  • les coefficients stoechiométriques de l'équation
Vous pouvez aussi choisir les quantités de matière dans l'état initial pour chacune des espèces chimiques (2e ligne). Lorsque ces réglages sont effectués :
  1. déplacer le curseur avancement de la reaction afin de re-calculer les quantités de matière pour un avancement x.
  2. une fois le curseur déplacé, cliquer sur VALIDER pour prendre une photographie du système chimique. Le diagramme plus bas donne une image des quantités de matière dans cet état d'avancement x

Rappel sur la définition d'avancement

On rappelle que le curseur est :
  • complétement à gauche pour l'état initial: x=0
  • complètement à droite pour l'état final: x = x(max)
  • dans une position intermédiaire pour un état intermédiaire: 0 < x < x(max)
